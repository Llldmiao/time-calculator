# 时间计算器 (Time Calculator)

这是一个简单的单页 Web 应用，用于执行常见的时间计算任务。

## 在线访问

您可以直接点击以下链接使用：

- **GitHub Pages**: [https://llldmiao.github.io/time-calculator/](https://llldmiao.github.io/time-calculator/)
- **Netlify**: [https://peaceful-sfogliatella-36014c.netlify.app/](https://peaceful-sfogliatella-36014c.netlify.app/)

## 功能

1. **计算时间差**：输入开始时间和结束时间，计算两者之间的间隔（天、小时、分钟）。
2. **时间推算**：输入基准时间，选择加或减去特定的时间量（天、小时、分钟），计算结果时间。

## 使用方法

1. **在线使用**：访问上方提供的任一链接。
2. **本地使用**：下载代码，直接在浏览器中打开 `index.html` 文件即可。无需安装任何依赖或服务器。

## 特点

- **零依赖**：纯原生 HTML/JS/CSS 实现，体积小巧。
- **主题切换**：支持明亮/暗黑模式一键切换，并自动记忆偏好。
- **响应式设计**：完美适配移动端和桌面端。
